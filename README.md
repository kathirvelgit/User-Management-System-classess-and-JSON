# 👨‍💻 JavaScript User Management & JSON Data Handling

A beginner-friendly JavaScript project demonstrating **Object-Oriented Programming (OOP)** using classes and **JSON data handling**. The project covers creating user objects, managing multiple users, checking adulthood, parsing JSON data into JavaScript objects, and converting objects back into JSON.

---

## 🚀 Features

### 👤 User Management System
- 🏗️ Create a `User` class using ES6 Classes
- 📱 Store user details (Username, Mobile Number, Age)
- 📄 Display user information using class methods
- ✅ Check whether a user is an adult (`age >= 18`)
- 📦 Store multiple users in an array
- 🔁 Loop through users and display details

### 📄 JSON Data Handling
- 📝 Create JSON data as a string
- 🔄 Convert JSON string into JavaScript objects using `JSON.parse()`
- 📋 Display user information using loops
- 📤 Convert JavaScript objects back into JSON using `JSON.stringify()`
